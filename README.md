# Игра "Gallows" ("Виселица")

Проект представляет консольное приложение для игры в "Виселицу". 
Игроку нужно отгадать слово за определённое количество попыток. 
При каждой ошибке человечек на виселице дорисовывается.

# Что было разработано

- Загрузка слов и сообщений из ресурсов проекта 
- Чтение слов из файла
- Получение случайного слова из словаря
- Валидация пользовательского ввода
- Подсчёт количества угаданных букв
- Два Singleton класса, потому что хотел закрепить на практике 
изученные паттерны GoF. После завершения проекта понял, что в данном 
проекте он не решает реальной проблемы и усложняет код.

# Используемые технологии

- Java 24
- Maven
- maven-jar-plugin

# Структура проекта

- src/main/java/.../Gallows
  - model
    - Dictionary.java - Класс-данное, содержащий словарь
  - service
    - MessageLoader.java - Класс-загрузчик для сообщений из ресурсов
    - WordsFromFileReader.java - Класс для загрузки словаря из ресурсов
  - util
    - DictionaryUtil.java - логика работы со словарём (случайное слово, готовность словаря)
    - GameLogic.java - логика игры (подсчёт угаданных букв, проверка победа)
  - view
    - View.java - отображение в консоль
  - GallowsGame.java - контроллер для всей игры
  - Main.java - главный класс для запуска игры (через контроллер)
- src/main/resources
  - messages.properties - файл, содержащий сообщения
  - RussianWords.txt - файл для слов, используемых для угадывания

# Запуск игры

1. Скачивание

```bash
git clone https://github.com/IJackDaniel/GallowsGame.git
cd GallowsGame
```
Или скачайте ZIP-архив с GitHub и распакуйте
2. Сборка и запуск

Вариант со сборкой JAR:
```bash
mvn clean package
java -jar target/Gallows-1.0-SNAPSHOT.jar
```

Вариант запуска через Maven (без сборки JAR)
```bash
mvn exec:java -Dexec.mainClass="ru.IJackDaniel.Gallows.Main"
```

Или просто запустите в IDE метод main() из класса Main.
